# 随心记便签

桌面便签应用，支持今日待办 + 随心记，每晚6点自动归档到 Word。

## 功能
- 📅 日期栏 + 日历（支持重点标记）
- ✅ 今日待办（拖拽排序、截止提醒、未完成自动迁移次日）
- 📝 随心记富文本编辑（粘贴图片、链接）
- 💾 每晚6点自动归档到 `D:\wenya\随心记\随心记归档.docx`
- 🔔 截止前3小时系统通知 + 便签内红色高亮
- ⚽ 收起为小球 / 最小化到系统托盘

## 开发启动

```bash
bash scripts/dev.sh
```

## 技术栈
- Electron 29 + Vue 3 + Pinia
- Tiptap v2（富文本编辑器）
- sql.js（SQLite，无需编译）
- node-cron（定时任务）
- docx（Word 归档）
