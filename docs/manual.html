🪟 智能浮窗

随心记悬浮在所有窗口上方，失焦后缩成屏幕边缘的小球；需要时点击小球立即唤出

  • 窗口失焦或点击 ● 按钮，主窗口隐藏，屏幕边缘出现紫色小球

  • 拖拽小球 自由调整位置
  • 若有逾期待办，小球右上角出现红色数字角标

📌 置顶

  • 点击标题栏右侧的 ↑ 按钮，按钮变蓝（如图所示）
  • 置顶后失焦不会缩成小球，窗口始终可见
  • 再次点击取消置顶

03 · DEADLINE

🕐 截止时间

给待办设置截止时间，临近时变黄警告，逾期后变红并高亮整行

设置方式

  • 点击待办右侧的 「截止时间」 文字，弹出时间选择器

05 · MIGRATE

手动迁移

1

自动迁移

  • 每天 00:01 自动将昨天未完成的待办迁移到今天
  • 应用关机跨越多天，重启时自动扫描过去 30 天补迁
  • 已手动删除的条目不会被重新迁移

06 · NOTES

📝 随心记笔记

窗口下半部分是自由书写的富文本区，支持标题、加粗、颜色、列表等完整格式化功能

格式工具栏

  • B / I / S：加粗、斜体、删除线
  • A：字号调整（放大 / 缩小）
  • •≡：圆点列表 / 数字编号列表
  • 🎨：字体颜色 + 荧光高亮
  • ✏️：格式刷（单击刷一次，双击持续刷）
  • ↺：清除格式（保留列表 / 全部取消）
  • ？：开关拼写检查
  • 💾：立即归档到 Word 文档

有序列表编号

  • 点击编号（1. 2. 3.）弹出菜单，可双击修改起始数字

右键复制

  • 选中文字后右键，弹出「复制 / 粘贴 / 全选」菜单

09 · HISTORY

📅 历史日期

点击标题栏日历图标，查看并跳转到任意历史日期的待办和笔记；也可一键批量打开最近 N 天

日历导航

  • 点击标题栏左侧的 📅 图标，弹出月历
  • 有内容的日期显示紫色圆点
  • 点击某一天跳转查看当天内容，点 ‹ › 切换月份

一键打开近 N 天

  • 顶部「打开前 － 3 ＋ 天」可调整天数
  • 点击 「一键打开」，自动从最近 N 天中找到有内容的日期依次打开
  • 适合快速回顾近几天的工作进展

每日归档

  • 每天 18:00 自动将随心记内容归档到本地 Word 文档
  • 也可点击随心记区右上角的 💾 立即归档
